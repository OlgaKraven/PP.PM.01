Модуль 2 — Разработка дизайна ИС (UI/UX)

Теория + встраиваемые интерактивные примеры
Форма авторизации Сетка Grid/Flex Модальное окно Слайдер A11y

Форма авторизации (UI + UX)

Меткованные поля, подсказки, валидация на клиенте, видимые ошибки, доступность с клавиатуры.

Подсказка: попробуйте отправить пустую форму и ввести короткий пароль, затем исправьте ошибку.

Адаптивная сетка (Grid + media queries)

Регулируйте ширину «макета» ползунком, чтобы увидеть перестройку колонок.

Модальное окно (доступное)

Откройте модалку, закройте клавишей Esc или кликом по затемнению. Фокус возвращается на кнопку.

Простой слайдер

Автопрокрутка с ручным управлением стрелками.

Фрагменты A11y

Проверка alt, ARIA‑меток и видимого фокуса.

Стили берутся из вашего styles.css для оболочки; каждое демо самодостаточно внутри своего iframe.
